# 🛋️ Davici Furniture Website

A modern and responsive e-commerce furniture website built using React.js. This project showcases a clean UI, reusable components, and smooth user experience.

---

## Live Demo
👉 https://davici-furniture.vercel.app/

---

##  Features
- Responsive design (Mobile + Desktop)
- Modern UI/UX design
- Product listing and display
- Reusable React components
- Smooth navigation
- Clean layout structure

---

##  Tech Stack
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Flexbox & Grid

---

##  Concepts Used
- Component-Based Architecture
- React Hooks (useState, useEffect)
- State Management (Basic)
- Responsive Web Design
- DOM Manipulation
