# 🛋️ Davici Furniture — Premium E-Commerce Store

A modern, fully responsive furniture e-commerce web app built with React.js and Tailwind CSS.

🔗 **Live Demo:** [davici-furniture.vercel.app](https://davici-furniture.vercel.app)

---

## ✨ Features

- 🛒 **Shopping Cart** — Add, remove, update quantity with localStorage persistence
- ❤️ **Wishlist** — Save favourite products across sessions
- 🔍 **Live Search** — Instant product search with overlay
- 🌙 **Dark Mode** — Full dark/light theme toggle
- 📦 **Product Detail Page** — Full page with related products
- 🔽 **Filter & Sort** — Filter by category, sort by price/rating
- 🎞️ **Animated Hero Carousel** — Auto-sliding with drag support
- 💫 **Framer Motion Animations** — Smooth page and component animations
- 🔔 **Toast Notifications** — Real-time feedback on all actions
- 📱 **Fully Responsive** — Mobile, tablet and desktop

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React.js | Frontend framework |
| Tailwind CSS | Utility-first styling |
| Framer Motion | Animations |
| React Router DOM | Client-side routing |
| React Hot Toast | Notifications |
| Lucide React | Icons |
| localStorage API | Cart & wishlist persistence |
